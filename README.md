# Random Gift Idea Generator

🎁 A responsive web application that helps users discover random and creative gift ideas instantly.

🔗 Live Demo:  
https://imritesh18.github.io/RandomGiftIdea/

---

## 📌 Overview

Random Gift Idea Generator is a fun and interactive application designed to generate unique gift suggestions for different occasions.

The platform allows users to:
- Generate random gift ideas
- Explore creative present suggestions
- Enjoy a clean and responsive interface
- Discover gifts quickly and easily

---

## 🚀 Features

- Random gift generation
- Responsive UI design
- Interactive user interface
- Beginner-friendly project
- Fast and lightweight application

## ▶️ How to Run Locally

### 1️⃣ Clone Repository

```bash
git clone https://github.com/imritesh18/RandomGiftIdea.git
```

### 2️⃣ Open Project Folder

```bash
cd RandomGiftIdea
```

### 3️⃣ Install Backend Dependencies

```bash
cd Backend
npm install
```

### 4️⃣ Start Backend Server

```bash
node server.js
```

### 5️⃣ Run Frontend

Open:

```text
index.html
```

in your browser.
